
# Day 7: Arrays

## Task
Given an array of integers, print the elements in reverse order as a single line of space-separated numbers.

## Input Format
- First line: Integer `n`, the size of the array.
- Second line: `n` space-separated integers.

## Output Format
- A single line of `n` space-separated integers in reverse order.

## Sample Input
4
1 4 3 2


## Sample Output
2341


## Concepts Covered
- Arrays
- Loops
- Reverse traversal

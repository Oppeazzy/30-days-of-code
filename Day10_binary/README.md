
# Day 10: Binary Numbers

## Task
Given an integer `n`, convert it to binary and print the maximum number of **consecutive `1`s** in its binary representation.


## Sample Input
13

## Sample Output
2

## Explanation
- Binary of `13` is `1101`
- Max streak of 1s = 2
